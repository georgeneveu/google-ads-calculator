# Google Ads Calculator (ROI)
Interactive Google Ads ROI Calculator. Estimates clicks, leads, customers, revenue, ROAS, and CPA based on budget, CPC, conversion rate, and AOV. Designed to help marketers quickly evaluate ad profitability with a clean, responsive UI and real-time calculations. Built with HTML, CSS, and JavaScript. 

It helps estimate the performance of Google Ads campaigns by calculating:

- Clicks
- Leads / conversions
- Customers
- Revenue
- Cost per acquisition (CPA)
- Return on Ad Spend (ROAS)

## 🔧 How it works
The calculator uses basic marketing formulas:

- Clicks = Budget / CPC  
- Leads = Clicks × Conversion Rate  
- Customers = Leads × Lead-to-Customer Rate  
- Revenue = Customers × Average Order Value  

All calculations update in real time as you adjust inputs.

## 🚀 Features
- Real-time calculations
- Scenario presets (Conservative, Realistic, Aggressive)
- Responsive UI
- Advanced settings toggle
- Clean Google Ads-style interface
